# 日历视图页面设计

## 概述

新增日历视图页面，支持月历/年历切换，以日历网格形式展示所有历史记录。

## 页面关系

```
历史记录页面 (保留)
    ↓ 点击「查看全部记录」
日历视图页面 (新增)
    ↓ 点击日期格子
底部抽屉 (当天记录)
```

### 历史记录页面修改

- 保留现有列表形式
- 只显示最近 10 条记录
- 底部新增「查看全部记录 →」入口，跳转日历视图

### 日历视图页面 (新增)

路径：`/pages/calendar/index.vue`

## 页面结构

## 顶部导航栏

### 元素

- 左箭头：上一月/年
- 年月文字：显示当前视图的年月，点击弹出选择器
- 右箭头：下一月/年
- 视图切换：「月」「年」两个按钮

### 交互

- 点击左/右箭头 → 切换上/下月或年
- 点击年月文字 → 弹出年月选择器（年份：当前年 ± 2年）
- 点击「月」/「年」按钮 → 切换视图模式

## 月历网格

### 格子内容

```
┌─────────────┐
│     15      │  日期数字
│    ● 3      │  有记录：圆点 + 数量
└─────────────┘
```

### 格子状态

| 状态 | 样式 |
|------|------|
| 今天 | 日期数字主色高亮、加粗 |
| 有记录 | 显示圆点 + 记录数量 |
| 无记录 | 只显示日期，灰色 |
| 非本月 | 灰色淡化（上月末/下月初补充格子）|
| 选中 | 边框高亮或背景变色 |

### 交互

- 点击格子 → 底部抽屉弹出当天记录
- 左右滑动网格 → 切换上/下月

## 底部抽屉

### 结构

```
┌──────────────────────────────────┐
│ ═════════                        │  拖拽指示条
│                                  │
│ 4月15日 · 星期二                  │  日期标题
│ 共 3 条记录                       │  统计
├──────────────────────────────────┤
│ [记录卡片1]                       │
│   09:30 · 成形 · 黄褐色           │
│                                  │
│ [记录卡片2]                       │  记录列表
│   14:20 · 软便 · 黄色             │
│                                  │
│ [记录卡片3]                       │
```

### 交互

- 向下滑动 → 关闭抽屉
- 点击记录卡片 → 跳转编辑页面
- 点击「添加记录」→ 跳转添加页面，预填选中日期

### 空状态

- 点击无记录的日期 → 抽屉显示「暂无记录」+ 添加按钮

## 年历视图

### 小月历卡片

- 显示月份名称
- 缩小的日历网格（仅显示日期和圆点标记）
- 底部显示当月记录总数

### 交互

- 点击小月历 → 切换到该月的月历视图
- 左右滑动 → 切换上/下一年

## 数据加载

### API 复用

使用现有 `api.getRecords` 接口：

```typescript
getRecords({
  baby_id: number,
  start_date?: number,  // 时间戳
  end_date?: number     // 时间戳
})
```

### 加载策略

| 视图 | 参数 | 说明 |
|------|------|------|
| 月历 | start_date=本月1日, end_date=本月末 | 加载当月所有记录 |
| 年历 | start_date=本年1月1日, end_date=本年12月31日 | 加载全年记录，前端聚合统计 |

### 缓存

- 已加载的月份数据缓存到 record store
- 切换月份时优先使用缓存

## 边界情况

### 空状态

- 当月无记录：网格正常显示，格子无圆点
- 点击无记录日期：抽屉显示空状态

### 网络错误

- 加载失败：显示重试按钮
- 离线模式：使用本地缓存

### 年份范围

- 可选年份：当前年份 ± 2 年

### 数据刷新

- 添加/编辑/删除记录后，返回日历页面自动刷新当月数据
